# 💿 SSPECTRA AUDIO LAB v1.0

> Developed & Secured by **PHANT044M // PHANTOM NETWORK**

**SSPECTRA AUDIO LAB** is an integrated desktop studio application designed for sound designers, music producers, and loop makers to bulk-manage audio metadata and deconstruct sound libraries with absolute cross-platform compatibility.

---

## ⚡ Core Systems

### 🏷️ Tag Studio (Metadata Forge)
- **RIFF LIST INFO Integration:** Full dual-tagging injection for `.wav` containers. Metadata displays seamlessly in Windows Explorer, VLC, FL Studio, Ableton Live, and Telegram.
- **Flawless FLAC & MP3 Sync:** Completely rebuilds Vorbis Comment blocks and ID3v2.3 tags without metadata corruption or offsets.
- **DPI-Aware Lab:** Drop preview area with automatic RGBA transparency PNG conversions.

### 📦 Deconstructor (Archive Sorter)
- **Hybrid Target Scan:** Queue both directory folders and `.zip` archives simultaneously.
- **Deep Recursive Walk:** Scans directory chains of any depth, applying rapid regular-expression categories.
- **MD5 Duplicate Filter:** Computes cryptographic hashes to automatically exclude identical duplicates from sample packs.
- **Auto-Cleanup:** Physically moves (cuts) or copies assets and purges empty directory leftovers automatically.
- **Content Inventory:** Generates detailed `.txt` text catalogs listing precise sound counts in each category.
